# Bell-States-using-Qiskit
This project demonstrates the generation and measurement of the four Bell states using Qiskit.  

The circuits are executed on: 
    - AerSimulator 
    - IBM Quantum Hardware  
    
Bell States: 
        - Φ⁺ (Phi Plus), 
        - Φ⁻ (Phi Minus), 
        - Ψ⁺ (Psi Plus) & 
        - Ψ⁻ (Psi Minus)
